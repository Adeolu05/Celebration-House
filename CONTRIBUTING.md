# Contributing to Celebration House

Thank you for your interest in contributing to the RCCG Region 32 Celebration House landing page!

## How to Contribute

1.  **Fork the repository**.
2.  **Create a new branch** for your feature or bug fix.
3.  **Make your changes**.
4.  **Ensure code quality**:
    *   Follow the existing code style.
    *   Run `npm run build` to verify the project builds.
5.  **Submit a Pull Request** with a clear description of your changes.

## Code of Conduct

Please be respectful and professional in all interactions within this project.
